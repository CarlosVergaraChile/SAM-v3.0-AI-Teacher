# Capacitacion_SENCE: Estructura de Curso OTEC

## Objetivo del Negocio
Diseñar e implementar un curso de capacitación certificado SENCE (Sistema Nacional de Capacitación y Empleo) que cumpla con estándares de calidad y relevancia laboral. Generar valor a participantes y validar competencias profesionales.

**Socios:** Sergio Orellana, Marcelo Castillo

## Stack Tecnológico Sugerido
- **Plataforma LMS:** Moodle SENCE-compatible
- **Contenido:** Módulos interactivos, evaluaciones prácticas
- **Backend:** Python Django
- **Base de Datos:** PostgreSQL
- **Certificación:** Integración con base de datos SENCE
- **Reporting:** Dashboard de seguimiento estudiantes

## Próximos Pasos Inmediatos
1. Validar estándares SENCE para el área de capacitación
2. Definir objetivos de aprendizaje medibles
3. Crear cronograma del curso (duración, número de horas)
4. Diseñar actividades prácticas y evaluaciones
5. Solicitar acreditación ante SENCE
