# Salud_Jeldres: Curso de Gestión de Operaciones en Salud

## Objetivo del Negocio
Desarrollar un curso integral de gestión de operaciones dirigido a profesionales del sector salud en Chile. El curso capacita en administración de recursos, optimización de procesos clínicos, y mejora continua en instituciones de salud.

**Cliente:** Sr. Jeldres

## Stack Tecnológico Sugerido
- **Plataforma LMS:** Moodle o Canvas
- **Contenido:** Módulos interactivos (video, documentos PDF, cuestionarios)
- **Backend:** Python Flask/Django
- **Base de Datos:** PostgreSQL
- **Hosting:** AWS o DigitalOcean
- **Analytics:** Google Analytics / Mixpanel

## Próximos Pasos Inmediatos
1. Definir estructura de módulos del curso (5-8 módulos)
2. Crear content outline con temas específicos
3. Identificar plataforma LMS preferida del cliente
4. Establecer cronograma de desarrollo (8-12 semanas)
5. Asignar equipo de contenido (creadores, instructores)
